## 책 선정 이유

 나는 대학교 3학년이다. 운영체제 과목도 수강 했다. 하지만 “운영체제란 무엇인가” 라는 질문에 “하드웨어와 사용자를 연결해주는 소프트웨어를 의미하고, 리눅스, 윈도우, 맥이 OS다.”라는 암기에 가까운 대답밖에 하지 못한다. 

책의 소개글에서 저자는  “개발자 혹은 개발자 지망생에게 필요한 전공 지식도”라 말한다. **혼자 공부하는 컴퓨터 구조+운영체제** 책을 통해 전공자로서의 CS 지식을 이해하고 깊이 있게 탐구하고자 한다. 애매모호하게 알고 있는 현재 지식을 보충하고 용어들을 유기적으로 엮어 전체 구조를 이해 하고자 한다.

## Chapter 1-1 컴퓨터 구조를 알아야 하는 이유

컴퓨터 구조는 실력 있는 개발자가 되려면 반드시 알아야 할 기본 지식이다.

#### Why?

1. 문제 해결 (프로그램이 실행 되지 않을 때, 그것은 코드의 문제가 아닐지도 모른다.)
2. Cost (개발 및 배포에서 발생 할 성능, 용량, 비용 고려)

## Chapter 1-2 컴퓨터 구조의 큰 그림

컴퓨터 구조란 컴퓨터가 이해하는 정보, 컴퓨터의 네 가지 핵심 부품으로 구성된다. 

- 컴퓨터가 이해하는 정보: 데이터, 명령어
- 컴퓨터의 네 가지 핵심 부품: CPU, 메모리, 보조기억장치, 입출력장치

마더보드 안에 CPU, 메모리가 있고 외부에 보조기억장치와 I/O장치가 있다. 마더보드와 외부와의 소통은 시스템 버스가 한다.

명령어가 들어오면 아래와 같은 순서로 처리된다. (해당 예시의 명령어: 더하라, 3번지와 4번지를)

1. 제어장치가 명령어를 읽어 들이기 위해 메모리에 ‘메모리 읽기’ 제어 신호를 보낸다.
2. 메모리는 해당 번지에 저장된 명령어를 CPU에게 보내주고, 레지스터에 명령어를 저장한다. 제어장치는 다시 3번지와 4번지의 데이터를 알기위해 ‘메모리 읽기’ 제어 신호를 보낸다.
3. 메모리는 다시 CPU에게 데이터를 보내주는데 이 데이터는 각각 다른 레지스터에 저장된다. ALU는 이를 계산하여 레지스터에 저장한다.
4. “이후 저장하라, 연산 결과를”이라는 명령어 처리 과정도 있지만, 이는 생략한다.

CPU 내에는 ALU, 제어장치, 레지스터가 존재하는데 레지스터와 메모리(주기억장치) 둘 다 저장하는 용도인 것 같은데. 어떻게 다를까?

*시스템 버스: 주소 버스(주소를 주고받는 통로), 데이터 버스(명령어와 데이터를 주고받는 통로), 제어 버스(제어 신호를 주고받는 통로)로 구성된다.

## Chapter 2-1 0과 1로 숫자를 표현하는 방법

컴퓨터가 표현하는 정보 단위와 이진법에 대해 공부한다.

### 정보 단위

정보 단위의 가장 작은 단위는 bit이다. 전구와 같다.

(bit → byte(8 bit) → KB(1024 B) → MB(1024 KB) → GB(1024 MB) → TB(1024 GB))

### 이진법

16진수는 왜 등장할까? → 2진접과 16진수는 서로 바꾸기 용이하다. 2진법의 긴 표현을 16진수의 짧은 표현으로 재정의 가능하다. 2^4)

십진수는 앞에 -를 붙여 음수를 표현한다. 그렇다면 이진수는 음수를 어떻게 표현할까? → 2의 보수로 표현한다. 2의 보수의 사전적 의미는 ‘어떤 수를 그보다 큰 2^n에서 뺀 값’을 의미한다. (특징: -(-A) = A인 것처럼, 음수를 두 번 구하면 제자리로 돌아간다.)

#### 그런데 이게 음수인지 양수인지 어떻게 알까?

플래그(Flag)를 사용하여 추가 표현한다. (4장에서 추가 설명) 그러면 보수 표현은 왜 하는걸까

## 추가 조사

1. CPU 내에는 ALU, 제어장치, 레지스터가 존재하는데 레지스터와 메모리(주기억장치) 둘 다 저장하는 용도인 것 같은데. 어떻게 다를까?
    
    ⇒ 둘 다 저장이지만. 위치, 속도, 용량, 역할이 완전히 다르다. 레지스터는 CPU 내부에 있는 저장 공간이다. ALU가 연산할 때 직접 접근하는 피연산자 저장소이다. 접근 속도가 CPU 클럭과 동일한 수준(1사이클 이내)이고 개수는 수십 개, 크기는 64비트 CPU라면 64비트짜리로 극소량이다.
    
    요약: 빠르지만 비싸다.
    
2. 플래그(Flag)를 사용하여 추가 표현한다. (4장에서 추가 설명) 그러면 보수 표현은 왜 하는걸까
